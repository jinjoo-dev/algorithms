# 알고리즘 문제 풀이

알고리즘 문제 풀이 저장소입니다. 백준, 구름, 프로그래머스 등 다양한 플랫폼의 문제들을 풀고 있습니다.

## 디렉토리 구조

```
algorithms/
├── baekjoon/          # 백준 온라인 저지
├── goorm/             # 구름 알고리즘
└── programmers/       # 프로그래머스
```

## 사용 언어

- **JavaScript** - 주 사용 언어로 알고리즘을 학습하기 위해 자바스크립트/Node.js 환경에서 문제를 풀고 있습니다.
- **Python** - 알고리즘 풀이에 효율적인 문법을 학습하기 위해 사용했으나, 현재는 주 사용 언어에 집중하고 있습니다.

## 파일 명 규칙

파일명은 `문제번호_문제제목.확장자` 형식을 따릅니다.

예시:

- `10989_수_정렬하기_3.py`
- `1157_단어_공부.js`
- `1260_DFS와_BFS.js`

## 주요 알고리즘 유형

이 저장소에는 다음과 같은 알고리즘 유형의 문제들이 포함되어 있습니다:

- 그래프 탐색 (DFS, BFS)
- 다이나믹 프로그래밍
- 그리디 알고리즘
- 정렬
- 이진 탐색
- 스택/큐/힙
- 트리
- 수학
- 문자열 처리
- 시뮬레이션
